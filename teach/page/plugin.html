-- Stage --

第1模块 -- 布局

模块描述

大布局：在得到效果图的时候，第一件事情，不是敲代码，而是构思整个页面的布局,大布局由外而内。

阶段须知

HTML 是一门什么样的语言？

看书-看入门书籍-《HTML与CSS入门经典_第9版》

搭建基本demo(模板)

                                        < !doctype html >          -- 决定浏览器的渲染模式[标准模式 | 怪异模式]
                                        <html>                           -- 这里是整个页面的开始，也是根节点
                                            <head> </head>    -- 编码、样式表引入、js引入（后面会讲到，JS引入其实放在body内部的最底下，
                                                                               其实是最合理的）、meta的各种效果控制
                                            <body> </body>    -- 这里是页面的展示内容，放在body的内容才是可见的
                                        </html>

html,head,body是html中最基础的3个元素, 而< !doctype html > 则决定浏览器的渲染方式（以后会提到）

常用标签？

块状

div、p、ul、ol、li、h1-h6 ....(HTML5:header、main、article....) 当然还有很多。只是用的相对较少

内联

span、a、em、i、、b、strong、font、img、lable........(当然这也不是全部)

内联块状

所有的表单元素

----------------------- 这只是分割线 -----------------------
table

table

这儿都是常用的，W3C上差不多有110个左右，当然也还有一些没公布出来的，但是那些几乎不会用到。
